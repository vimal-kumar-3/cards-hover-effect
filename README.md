#  Cards Hover Effect (HTML + CSS)

A stylish card UI animation project where card images scale and react on hover.  
This project was built using **HTML** and **CSS** to practice transitions, transforms, and layout design.

---

##  Features
- Four card layout  
- Hover animation  
- Smooth scaling effect  
- Image-based card UI  
- Beginner-friendly code

---

##  Tech Stack
- HTML5  
- CSS3
